Nocturne API v1

Gestionnaire de containers distants.

Présentation

Je vous présente aujourd'hui le projet Nocturne API. Ce projet est un gestionnaire de containers distants. Il vous permet d'envoyer, de gérer et de recevoir des fichiers hébergés sur nos serveurs.

Intégrer cette API dans vos programmes vous sera utile si vous avez besoin de stocker des informations diverses (utilisateurs, images, etc.).

Vous pourrez gérer votre container personnel et partager des ressources au monde entier via le container collectif. Il vous suffira de spécifier si l'import est privé ou public.

Les données stockées sont chiffrées et protégées. À chaque upload, la donnée va être chiffrée puis déchiffrée en cas de demande. Vous n'aurez donc pas à vous soucier de la lisibilité de vos fichiers.

En cas de besoins, votre clé pourra être réinitialisée. Vous pourrez transférer vos données de containers en containers.

Fonctionnalités

Ce projet vous permettra de stocker vos données en utilisant des requêtes HTTP/s. De ce fait, vous pourrez intéragir avec votre container via votre propre interface et/ou projet.

Les fonctionnalités actuelles suivent les standards HTTP/s.

  • POST vous permettra de poster une donnée,
  • GET vous permettra de récupérer une donnée,
  • PUT vous permettra de modifier une donnée,
  • DELETE vous permettra de supprimer une donnée.

    • L'exemple le plus prometteur d'utilisation de vos containers seraient pour la création d'applications NodeJS qui utilisent la technologie Electron. Vous pourrez alors communiquer avec vos containers via une interface.
